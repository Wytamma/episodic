# Guides

This section contains reproducible, study-specific walkthroughs that recreate analyses from published studies using `episodic`.

Guides are designed to help you:

- understand the biological and modeling context of a study
- run the same clock-model comparisons with `episodic`
- reproduce key outputs (model-comparison summaries, MCC trees, and plots)
- compare your run outputs against expected study-style results

## What each guide includes

Each guide typically provides:

- study reference and analysis scope
- input data and configuration used
- runnable commands
- expected output artifacts to inspect
- reproducibility notes (versions, runtime, environment)

Use these guides as practical templates for applying Episodic to your own datasets.
